# Build a Complete Khatabook Clone (Production Ready)

## Project Goal

Create a modern, professional, mobile-first accounting application similar to Khatabook.

The application is for small shop owners, wholesalers, retailers, manufacturers, and business owners who are not highly educated. The UI must be extremely simple, easy to understand, fast, and require as few taps as possible.

The design should feel very close to Khatabook while using original code and assets.

Write production-quality code.

Use custom hooks.

Use Firebase best practices.

Use Firestore transactions where necessary.

Use proper error handling.

Use loading states.

Use toast notifications.

Create a scalable folder structure.

Generate clean, maintainable code with comments where necessary.

The final application should look and feel like a premium business accounting app similar to Khatabook while maintaining original implementation. It should be extremely simple for shop owners with little technical knowledge, highly optimized for mobile devices, fully responsive, installable as a PWA, and powered entirely by Firebase Authentication and Firestore.
